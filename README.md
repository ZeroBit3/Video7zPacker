# Auto 7z Interactive Packer

这是一个基于 Python 的交互式自动化分卷加密打包工具。它调用系统的 7-Zip 命令，自动识别视频集数，并将大文件分卷压缩加密。

## 功能特性
- **交互式配置**：自动询问密码和输出路径。
- **智能识别**：自动从文件名解析集数（如 E01, 第1集）。
- **自动分卷**：超过 1.8GB （这个阈值大小自有用意，很多事情会方便不少）的文件自动按 1805MB （小于1.8GB）分卷（兼容 FAT32/网盘限制）。
- **安全加密**：使用 7z 格式并开启文件名加密。

## 环境要求
1. **Python 3.x**
2. **7-Zip**: 必须安装 7-Zip 并将 `7z.exe` 所在的目录添加到系统的 `PATH` 环境变量中。

## 使用方法
```bash
python pack_7z.py
```

## 注意事项
* 脚本默认在当前目录下扫描视频文件。
* 确保输出目录有足够的写入权限。
